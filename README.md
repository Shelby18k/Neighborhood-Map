This project is the final project in the FEND

## Instructions
- Clone this repo
- Then run command

## Dependencies
- React responsive module
### `npm install react-responsive`
- Google Maps
- FourSquare API
- React FourSquare API module
### `npm install react-native-foursquare-api`
